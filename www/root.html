You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /mulitply -- returns the multiplication of two numbers
  • /github -- returns an abbreviated JSON of a users GitHub repo
  • /rickandmorty -- get rick and morty lines. May contain foul language!
    • Params: paragraphs={num} quotes={num}
    • Example: /rickandmorty?paragraphs=2&quotes=2

      • Returns paragraphs of text to use instead of lorem ipsum

  • /ageguesser -- guess the average age of your name.
    • Params: name={name} age={num}
    • Example: /ageguesser?name=bella&age=32

      • Use demographics to guess the age of a person by name alone. Returns the percentage difference between the actual age and demographic based guess.
